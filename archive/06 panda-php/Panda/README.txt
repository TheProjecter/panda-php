Panda PHP Components
===============================================================================

About

PandaPHP is an open source library of handy PHP scripts. It is not a
full-stack framework like CakePHP, or CodeIgniter; instead we'll describe it
as a "component library" or simply "library".

If you're looking for a framework, PandaPHP can still help you! We have also
developed a framework in PandaPHP.
